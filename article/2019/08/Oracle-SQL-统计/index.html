Oracle中的 ‘空字符串’ 和 ‘NULL’ 的区别？

'' ： ‘空字符串’

' ' ：不是 ‘空字符串’ ，中间有空格；

‘空字符串’ 和 ‘NULL’ 不同。例如：where条件中 is null 和 ='' 结果不一样。

我们无法将 ‘空字符串’ 存进 Oracle 数据库。

感觉有 ‘NULL’ 了，’空字符串’ 有点多余。
