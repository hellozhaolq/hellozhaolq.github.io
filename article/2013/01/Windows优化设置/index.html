Windows优化设置

发表于 2013-01-06

Win10

设置 >>> 系统 >>> 电源和睡眠，进入睡眠状态改为从不。

设置 >>> 隐私 >>> 常规，关闭所有隐私选项。

设置 >>> 隐私 >>> 诊断和反馈，反馈频率改为从不。

设置 >>> 应用 >>> 离线地图，关闭自动更新地图。

设置 >>> 更新和安全 >>> 传递优化，关闭允许从其他电脑下载。

设置 >>> 网络和 Internet >>> Windows 防火墙，合理设置。

可选操作：

设置 >>> 隐私 >>> 后台应用，关闭应用后台运行。

右键磁盘 >>> 属性 >>> 工具 >>> 优化 >>> 更改设置，频率改为每月。

右键此电脑 >>> 属性 >>> 高级系统设置 >>> 环境变量，用户变量和系统变量中的 TEMP 和 TMP 可改为富裕的磁盘位置。

右键C盘 >>> 磁盘清理，可清理更新后的残余。
