Javaеӣһи°ғжңәеҲ¶

еҸ‘иЎЁдәҺ 2015-11-03 еҲҶзұ»дәҺ Java

жҺЁиҚҗй“ҫжҺҘ

Javaдёӯзҡ„еӣһи°ғ

жүҖи°“еӣһи°ғ

жүҖи°“еӣһи°ғпјҢе°ұжҳҜAзұ»и°ғз”ЁBзұ»дёӯзҡ„жҹҗдёӘж–№жі•CпјҢз„¶еҗҺBзұ»еҸҚиҝҮжқҘи°ғз”ЁAзұ»дёӯзҡ„жҹҗдёӘж–№жі•DпјҢиҝҷдёӘDж–№жі•е°ұеҸ«еӣһи°ғж–№жі•гҖӮ

ејӮжӯҘ+еӣһи°ғ

гҖҖгҖҖжңүдёҖеӨ©е°ҸзҺӢйҒҮеҲ°дёҖдёӘеҫҲйҡҫзҡ„й—®йўҳпјҢй—®йўҳжҳҜвҖң1 + 1 = ?вҖқпјҢе°ұжү“з”өиҜқй—®е°ҸжқҺпјҢе°ҸжқҺдёҖдёӢеӯҗд№ҹдёҚзҹҘйҒ“пјҢе°ұи·ҹе°ҸзҺӢиҜҙпјҢзӯүжҲ‘еҠһе®ҢжүӢдёҠзҡ„дәӢжғ…пјҢе°ұеҺ»жғіжғізӯ”жЎҲпјҢе°ҸзҺӢд№ҹдёҚдјҡеӮ»еӮ»зҡ„жӢҝзқҖз”өиҜқеҺ»зӯүе°ҸжқҺзҡ„зӯ”жЎҲеҗ§пјҢдәҺжҳҜе°ҸзҺӢе°ұеҜ№е°ҸжқҺиҜҙпјҢжҲ‘иҝҳиҰҒеҺ»йҖӣиЎ—пјҢдҪ зҹҘйҒ“дәҶзӯ”жЎҲе°ұжү“жҲ‘з”өиҜқе‘ҠиҜүжҲ‘пјҢдәҺжҳҜжҢӮдәҶз”өиҜқпјҢиҮӘе·ұеҠһиҮӘе·ұзҡ„дәӢжғ…пјҢиҝҮдәҶдёҖдёӘе°Ҹж—¶пјҢе°ҸжқҺжү“дәҶе°ҸзҺӢзҡ„з”өиҜқпјҢе‘ҠиҜүд»–зӯ”жЎҲжҳҜ2

ж–№ејҸдёҖ(з»Ҹе…ё)

3еӨ§еүҚжҸҗпјҡ
гҖҖгҖҖеүҚжҸҗ1пјҡClass Aе®һзҺ°еӣһи°ғжҺҘеҸЈCallBack
гҖҖгҖҖеүҚжҸҗ2пјҡclass AдёӯеҢ…еҗ«дёҖдёӘclass Bзҡ„еј•з”Ё
гҖҖгҖҖеүҚжҸҗ3пјҡclass BжңүдёҖдёӘеҸӮж•°жҳҜжҺҘеҸЈCallbackзұ»еһӢзҡ„ж–№жі• fun(CallBack callback, вҖҰ)
2еӨ§жӯҘйӘӨпјҡ
гҖҖгҖҖз¬¬дёҖжӯҘпјҡеҲ©з”ЁClass AеҢ…еҗ«зҡ„class Bеј•з”ЁеҺ»и°ғз”Ёclass Bзҡ„ж–№жі• fun(CallBack callback, вҖҰ) вҖ”вҖ”> Aзұ»и°ғз”ЁBзұ»зҡ„жҹҗдёӘж–№жі•C
гҖҖгҖҖз¬¬дәҢжӯҘпјҡclass BеңЁ fun(CallBack callback, вҖҰ) ж–№жі•дёӯи°ғз”ЁClass Aе®һзҺ°зҡ„жҺҘеҸЈж–№жі• вҖ”вҖ”> Bзұ»и°ғз”ЁAзұ»зҡ„жҹҗдёӘж–№жі•D

ж–№ејҸдәҢ

еүҚжҸҗпјҡ

гҖҖгҖҖclass BжңүдёҖдёӘеҸӮж•°жҳҜжҺҘеҸЈCallbackзұ»еһӢзҡ„ж–№жі• fun(CallBack callback, вҖҰ)пјҢе№¶дё”class BеҶ…йғЁе®ҡд№үдәҶеӣһи°ғжҺҘеҸЈCallBackгҖӮ
жӯҘйӘӨпјҡ
гҖҖгҖҖз¬¬дёҖжӯҘпјҡclass AдёӯиҺ·еҸ–пјҲжҲ–еҲӣе»әпјүclass Bзҡ„еј•з”ЁгҖӮ
гҖҖгҖҖз¬¬дәҢжӯҘпјҡclass Bзҡ„еј•з”Ёи°ғз”Ёclass Bзҡ„ж–№жі• fun(CallBack callback, вҖҰ)пјҢCallBackзұ»еһӢзҡ„еҸӮж•°жҳҜеҢҝеҗҚеҶ…йғЁзұ»
